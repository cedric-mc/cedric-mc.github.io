Projets de programmation

Game : Battle Of Balls

Python

The aim of this project was to create a ball placement game. Each user plays with one color. The aim of the game is to occupy the largest area with your color. Players take turns controlling the mouse, and the number of turns is decided in advance. The project was to be carried out by two people in 3 phases.
Game :
Documentation :

Battle Of Balls, programmed in Python, is played by 2 players

The aim of the game is simple : occupy as much space as possible on the playing area with your color.
You'll be able to select variants to spice up your game, and a color pairing to decide your color as a player, choose a nickname for the player (a default nickname is available if you don't enter anything), and enter the number of rounds (default minimum 5).

Battle of the Balls variations

You can play with 6 different variants : le Sablier, les Scores, la Taille de Boules, la Version Dynamique, la Terminaison et les Obstacles.
